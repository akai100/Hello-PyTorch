**transpose** 不是拷贝，而是改 stride**

## 调用 ```Tensor.t```方法实现转置

```python3
import torch

x = torch.randn(2, 3)
y = x.t()

x[0][0] = 0
```

```t()```实现矩阵转置，```y```不拷贝```x````的数据，共享同一份数据，只修改```stride````。修改```x```同时会修改```y```。

## 调用```Tensor.transpose```实现转置

```python3
def transpose(self, dim0: _int, dim1: _int) -> Tensor:
    pass
```

**交换张量两个指定维度**的方法，返回的是原张量的视图（View）（而非拷贝）：

+ 视图特性：转置后的张量与原张量共享内存，修改其中一个会影响另一个（节省内存）

+ dim0/dim1：要交换的两个维度的索引（从 0 开始，支持负索引：-1 = 最后一维，-2 = 倒数第二维）
