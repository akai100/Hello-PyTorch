## 1️⃣ 基本概念

```Tensor.t()``` 是 PyTorch 中 **二维张量（矩阵）** 的 **转置操作**。它的作用是把张量的行和列交换。

⚠️ 注意：t() **只适用于二维张量**（形状为 [m, n]），如果张量维度不是 2，会报错。

## 2️⃣ 语法

```python3
tensor.t()
```

+ **返回值**：返回一个新的张量，是原张量的转置视图（不拷贝数据）；

+ **原张量**不会被改变，除非你用 ```tensor = tensor.t()``` 重新赋值；

## 3️⃣ 示例

```python3
import torch

# 创建一个 2x3 的张量
x = torch.tensor([[1, 2, 3],
                  [4, 5, 6]])

print("原张量 x:")
print(x)
print(x.shape)

# 转置
y = x.t()
print("转置后的张量 y:")
print(y)
print(y.shape)
```

## 4️⃣ 和 ```.transpose()``` 的区别

+ ```t()```：只能用于二维张量，快捷方式；

+ ```transpose(dim0, dim1)```：适用于任意维度张量，可以交换指定的两个维度；
